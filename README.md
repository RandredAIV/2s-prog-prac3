# RU

# Алгоритмы и итераторы

## Описание

Данная работа посвящена изучению стандартной библиотеки алгоритмов C++ 
и работе с итераторами STL.

В рамках работы реализованы 10 самостоятельных программ, 
демонстрирующих использование контейнеров, алгоритмов и итераторов.

Каждая задача реализована в отдельном файле и компилируется отдельно.

---

## Цели работы

- Изучение библиотеки алгоритмов STL
- Изучение работы с итераторами
- Использование контейнеров vector и map
- Применение алгоритмов shuffle, count, count_if, accumulate, for_each, all_of, any_of, find, find_if
- Закрепление навыков программирования на C++

---

## Реализованные задачи

### Задача 1 — Перемешивание карт
Формирование колоды из 52 карт, перемешивание с использованием shuffle,
подсчет количества карт по мастям.

### Задача 2 — Крестики-нолики
Игровое поле 3x3, проверка победы с использованием алгоритмов STL,
подсчет символов с помощью count.

### Задача 3 — Угадывание числа
Генерация случайного числа, подсказки пользователю,
подсчет количества попыток.

### Задача 4 — Виселица
Выбор случайного слова,
проверка введенных букв с использованием find и find_if.

### Задача 5 — Блек-джек
Моделирование карточной игры,
подсчет очков с использованием for_each,
работа с итераторами.

### Задача 6 — Казнь
Случайный выбор студентов,
использование random_shuffle и count_if.

### Задача 7 — Штурм 502 кабинета
Генерация количества студентов по дням,
подсчет общего количества с помощью accumulate.

### Задача 8 — Ограбление
Расчет общей суммы долга,
использование fill_n и accumulate.

### Задача 9 — Рейтинг студентов
Вычисление среднего балла,
поиск студента с максимальным значением.

### Задача 10 — Голодные игры
Генерация количества студентов в группах,
подсчет среднего значения.

# EN

# Algorithms and Iterators

## Description

This project focuses on studying the C++ Standard Template Library (STL)
and working with iterators.

Ten independent programs were implemented,
each demonstrating the use of containers, algorithms, and iterators.

Each task is implemented in a separate file and must be compiled separately.

---

## Objectives

- Study STL algorithms
- Work with iterators
- Use vector and map containers
- Apply algorithms such as shuffle, count, count_if, accumulate, for_each, all_of, any_of, find, find_if
- Improve C++ programming skills

---

## Implemented Tasks

### Task 1 — Card Shuffling
Create a 52-card deck, shuffle it using shuffle,
count cards by suit.

### Task 2 — Tic-Tac-Toe
3x3 board, win checking using STL algorithms,
symbol counting with count.

### Task 3 — Number Guessing Game
Random number generation,
attempt counter.

### Task 4 — Hangman
Random word selection,
letter checking using find and find_if.

### Task 5 — Blackjack
Card game simulation,
point counting using for_each,
iterator usage.

### Task 6 — Execution
Random student selection,
use of random_shuffle and count_if.

### Task 7 — Classroom Queue
Daily student statistics,
total count using accumulate.

### Task 8 — Robbery
Debt calculation,
use of fill_n and accumulate.

### Task 9 — Student Rating
Average score calculation,
finding the highest average.

### Task 10 — Hunger Games
Group size simulation,
average value calculation.
